C++中多个对象共享虚函数表

Posted on 2019-03-05

今天有个同学问我一个有关C++中的虚函数问题，“C++中同一个类下面的多个对象是否共享一个虚函数表”。
先说答案，答案肯定是同一个类下面的多个对象共享一个虚函数表。
由于印象不够深刻，担心误导同学，我搜了下资料，发现有些说是共享有些说不共享。 talk is easy, show me the code. 不如写个很简单的代码验证下。 

#include <iostream>
using namespace std;

class Base {
public:
    virtual void Base_print() {
        cout << "Base print" << endl;
    }
};

int main(int argc, char ** argv) {
    Base a[3];
    for (int i = 0; i < 3; i++) {
        cout << i << " vtbl: " << *(int *)(&a[i]) << endl;
        cout << i << " Base_print: " << *(int *)*(int *)(&a[i]) << endl;
    }
    return 0;
}
